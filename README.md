# Student Performance Analysis Using NumPy

## Project Overview

This project demonstrates how NumPy can be used for data loading, filtering, statistical analysis, and performance evaluation using a student dataset.

The dataset contains information such as:

* Student ID
* Age
* Attendance
* Study Hours
* Marks

## Objectives

The main objectives of this project are:

* Load a CSV dataset using NumPy
* Analyze student performance
* Calculate statistical measures
* Filter and sort data
* Generate meaningful insights from the dataset

## Analysis Performed

* Total Number of Students
* Average Age
* Average Attendance
* Average Marks
* Highest Marks
* Lowest Marks
* Students Scoring Above 80
* Students Scoring Above 90
* Students Scoring Below 40
* Top 10 Marks
* Pass Percentage
* Students with Attendance Below 75%
* Students Studying More Than 8 Hours

## Technologies Used

* Python
* NumPy
* Google Colab

## Learning Outcomes

Through this project, I learned:

* Working with NumPy arrays
* Loading CSV files using NumPy
* Array indexing and slicing
* Boolean filtering
* Aggregate functions such as mean(), max(), min()
* Sorting and analyzing large datasets
